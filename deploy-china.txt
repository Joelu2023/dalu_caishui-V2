大陆财税工作室 — 国内访问（广州 COS）一键部署说明
================================================

一、为什么 Vercel 大哥打不开？
  海外节点，大陆手机/微信内浏览器经常超时。放到腾讯云 COS（广州）后，华南访问会稳定很多。

二、我已在项目里加好「一键上传」脚本（你只需配密钥 + 建桶）

  1. 打开 https://console.cloud.tencent.com/cos ，注册/登录。
  2. 新建存储桶：
     - 地域：广州 ap-guangzhou
     - 访问权限：可先「公有读私有写」（便于大哥直接打开网页）
     - 其他默认即可
  3. 打开 https://console.cloud.tencent.com/cam/capi ，创建「API 密钥」，得到 SecretId、SecretKey（勿泄露）。
  4. 在本项目根目录复制 .env.example 为 .env，填入：
       COS_SECRET_ID、COS_SECRET_KEY
       COS_BUCKET=你的桶名（控制台里完整名称，含 -APPID 后缀）
       COS_REGION=ap-guangzhou
  5. 在项目目录执行：
       npm run deploy:cos:full
     会自动：build → 上传 → 用 API 开启静态网站（索引/错误页均为 index.html）。
     若已开启过静态网站，仅更新文件可：npm run deploy:cos
     仅补开静态网站可：npm run cos:website
  6. （可选）若未用脚本，可到 COS 控制台该桶 → 基础配置 → 静态网站：
       开启；索引文档 index.html；错误文档填 index.html（单页路由需要）。
  7. 把控制台「静态网站」里给出的访问域名（cos-website...）发给大哥，
     用手机自带浏览器或华为浏览器打开（不要只依赖微信里点链接）。

三、Gitee Pages（备选）
  若不用 COS，仍可按此前说明用 Gitee；子路径部署需设置 VITE_BASE 后再 build:china。

四、微信打不开的补充
  即使已部署在国内，也建议「复制链接 → 系统浏览器打开」。
